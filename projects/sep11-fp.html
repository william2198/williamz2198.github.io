Freedom Project

The freedom project was the javascript year-long project that I worked using THREE.js on a 3D world parkour game. My project had around 300 lines of code that included the camera, object and the physics/ gravity. Some examples of javascript that I learned which I used in my project were for loops for the repeating buildings and clouds, and arrays to make sure every object that was in the array would apply gravity and shadow.

For my project, I had to use many tools to help me with my project such as youtube tutorials, forums and documentations. There were times where I would learn code about THREE.js and it would not be used in the final project. There were also multiple literations of the project. The first one had a third person camera instead of a first person camera. Even though there would a lot of struggles with the process, I am happy with how the project turned out.
